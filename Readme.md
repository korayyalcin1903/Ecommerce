# E-Commerce Site

This project is an e-commerce site that I developed using ASP.NET MVC. In the project, users can view products, add them to their basket and buy them. Product, category and user management can be done with the admin panel.

## Features

- **Product Management**: Adding, deleting, editing and filtering products by category with search box.
- **Order Management**: Listing, editing and deleting orders.
- Category Management**: Adding and deleting categories.
- **User Management**: Adding and deleting users.
- **User Login and Registration**: Users can register and login.
- **Admin Panel**: All management operations with the admin panel.
- **Payment System**: Payment integration with Iyzico.

## Technologies Used

- HTML
- CSS
- Bootstrap
- ASP.NET MVC
- Entity Framework
- MySQL
- JavaScript

## Installation

1. **Download Project Files**

    ```bash
    git clone https://github.com/korayyalcin1903/Ecommerce.git
    ```

2. **Install Required Packages**

    ```bash
    dotnet restore
    ```

3. **Set Database Settings**

    Make database connection settings in `appsettings.json` file.

4. **Update Database**

    ```bash
    dotnet ef database update
    ```

5. **Run the Project**

    ```bash
    dotnet run
    ```
